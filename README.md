# 离线作品媒体中心

单页离线媒体浏览器，用浏览器打开即可浏览本地作品目录，支持静态图、实况照片（短视频）、长视频和背景音乐。

## 快速开始

双击 `ai_studio_code.html` 在 Chrome / Edge 中打开，点击「选择素材总目录」选择本地文件夹即可。

## 目录结构要求

素材按作者分目录存放，目录名格式：

```
作者名-xxx-xxx-xxx-作品名/
```

具体规则：文件名 / 目录名按 `-` 分割后，第 5 段（索引 4）为**作者名**，不足 5 段则用父目录名兜底。

同一目录下的所有文件归为同一位作者。每个目录可包含一个音频文件作为背景音乐（支持 mp3 / wav / flac / aac / ogg 等 16 种格式）。

### 内容分类

| 类型 | 条件 | 呈现方式 |
| --- | --- | --- |
| 长视频 | 时长 > 5s | 独立作品，支持播放/暂停、进度条、连播 |
| 实况照片 | 时长 ≤ 5s | 归入图集，与静态图片混合轮播 |
| 静态图片 | — | 单张或图集轮播（按 `_数字` 后缀去重分组） |

## 导航方式

| 操作 | 动作 |
| --- | --- |
| 滚轮 / W / S / ↑ / ↓ | 切换上一个 / 下一个作品 |
| 鼠标拖拽（>50px） | 同上 |
| A / D / ← / → / 左右箭头按钮 | 图集中切上一张 / 下一张；视频中快退/快进 5s |
| 右侧头像按钮 / F 键 | 打开 / 关闭作者主页抽屉 |

## 播放控制

| 操作 | 静态图片 | 实况照片 | 背景音乐 |
| --- | --- | --- | --- |
| 空格键 | 暂停/恢复轮播 | 播放/暂停视频 | 同步启停 |
| 点击内容区 | 暂停/恢复轮播 | 播放/暂停视频 | 同步启停 |
| 中间暂停按钮 | 暂停/恢复轮播 | — | 同步启停 |
| M 键 | — | — | 静音切换 |

- 暂停状态下切换到新作品会自动恢复播放。
- 同作品内切换子项保持暂停状态不变。

## 键盘快捷键

| 键 | 功能 |
| --- | --- |
| W / ↑ | 上一个作品 |
| S / ↓ | 下一个作品 |
| A / ← | 上一张（视频快退 5s） |
| D / → | 下一张（视频快进 5s） |
| 空格 | 暂停 / 恢复 |
| F | 作者主页抽屉 |
| M | 静音切换 |

## UI 布局

- **背景**：当前作品高斯模糊
- **左下角**：作者名、作品描述（支持 `#tag` 高亮）、音频文件名及播放状态
- **底部栏**：图集轮播指示点、进度条、时间 / 页码显示、静音、连播开关
- **右侧**：作者头像按钮（打开作者主页抽屉）
- **作者主页**：当前作者所有作品缩略图网格，点击跳转
